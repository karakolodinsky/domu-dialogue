# domudialogue
is a web application to be used for creation of simple [ADV](https://towardstheendsky.blogspot.com/2022/10/nvl-vs-adv-anatomy-of-visual-novels.html) style visual novel style scenes.

this application has been developed in vanilla html, css, and javascript.

## folder identification

### example_scene: 
an example scene produced by the logic of the tool
to be replaced with actual product when gui is finished

### tokenization_tester: 
a tool for testing tokenization of program commands.
type commands into the input, and the display of the scene will change.
in later iterations of the program, these commands will exist in a queue and execute on player advancement.

### script_queue: 
UNFINISHED TOOL.
used for creating the queue used for the scene.
a user will be able to input command and it will be added into the scene's script, create an iframe with the current script object to test how the scene will operate, delete commands from the script, move them around, and export the scene.
